```js

/>
```
## Props

<br />

| prop              |  type            | default  | description                                  |
| ----------------- | :-------------- :| :------: | :------------------------------------------- |
| title             | string           |  -    | Row title                    |
| component         | React element or string           |  -       | Row description                     |
